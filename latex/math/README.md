#CBSE Mathematics Question Paper (LaTeX)

This repository contains a LaTeX source file to generate a CBSE Mathematics Question Paper. The document is structured for Section C of the exam, including multiple questions with figures and formatted answers.

Features

Professional formatting using article class.

Includes student information header with logo.

Uses LaTeX packages for mathematics, graphics, and custom lists:

amsmath, amssymb — for mathematical formulas.

graphicx — to include images like logo and figures.

geometry — to set page margins.

enumitem — for custom numbering of questions.

Supports inclusion of diagrams for questions.

Ready to compile into a PDF using pdflatex.
