#Project Name:Multi-Client TCP Chat Application using Sockets in C

#1.The main goal is:

To build a system where multiple users can communicate with each other in real
-time using network programming.

#2. Why this project is used
 This project is used to:

Send messages between users over a network
Simulate real-world chat systems
Understand how communication happens between devices

 Example:
WhatsApp
Telegram
Discord

Your project is a basic version of these apps


#3.Language used
 C programming language

#4.Why this project is important

This project teaches important concepts:

1. Networking
How two systems communicate
Client-server architecture
2. Socket Programming
socket(), bind(), connect(), send(), recv()
3. Multithreading
Handling multiple users at same time
4. Synchronization
Using mutex to avoid data conflicts
5. Protocol Design
Struct-based message communication

 Simple line:

This project is important because it demonstrates real-time communication,
 networking, multithreading, and system-level programming.
